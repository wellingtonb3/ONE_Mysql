# 🧠 Projeto MySQL — Insight Places

📘 *Módulo do Curso ONE | Oracle Next Education*

---

## 🚀 Visão Geral

Este repositório documenta o desenvolvimento do **banco de dados da Insight Places**, projeto prático do curso **#ONE Oracle Next Education**.
O foco foi compreender o ciclo completo do **MySQL**, desde a instalação até consultas e operações avançadas, aplicadas a um contexto de **análise de marketing e dados empresariais**.

---

## 🧩 Tecnologias e Ferramentas Utilizadas

* 🐬 **MySQL**
* 💻 **MySQL Workbench**
* 🔢 **Linha de comando (CLI)**

---

## 📚 Conteúdos Aprendidos

### 🏁 Aula 1 — Fundamentos e Instalação

* Conhecimento do projeto **Insight Places**
* Instalação e configuração do **MySQL**
* Exploração da interface **MySQL Workbench**
* Acesso ao banco via **linha de comando**

---

### 🧱 Aula 2 — Criação de Banco e Tabelas

* Criação de **bancos de dados** (`CREATE DATABASE`)
* Criação de **tabelas** (`CREATE TABLE`)
* Inserção de dados (`INSERT INTO`)
* Exploração de **colunas e tipos de dados**

---

### 🔑 Aula 3 — Estrutura e Tipos de Dados

* Identificação dos **tipos de dados** no MySQL
* Entendimento sobre **chaves primárias**
* Acesso e análise do **modelo físico** do banco de dados

---

### 🔍 Aula 4 — Consultas e Funções

* Uso de **operadores lógicos** e **de comparação**
* Criação de consultas com **funções AVG e COUNT**
* Aplicação de **JOINs** para relacionar tabelas
* Uso de **funções para dados do tipo DATE**

---

## 🧮 Resumo Final

Durante o projeto da **Insight Places**, foi desenvolvido um banco de dados completo, migrando informações reais para o **MySQL**.
As etapas incluíram:

✅ **Instalação e configuração** do ambiente
✅ **Criação e estruturação** das tabelas e relacionamentos
✅ **Inserção e manipulação** dos dados empresariais
✅ **Consultas analíticas** para gerar insights de marketing
✅ **Otimização e manutenção** da base de dados

O resultado é um sistema sólido e funcional, capaz de apoiar decisões estratégicas baseadas em dados.

---

## 🌟 Conclusão

Este projeto proporcionou uma visão prática e integrada do **uso do MySQL em ambientes reais**, desenvolvendo habilidades essenciais em **SQL, modelagem e análise de dados**.

Agora, o aprendizado serve como base para atuar em projetos de **Data Analysis, Business Intelligence** e **engenharia de dados** com confiança.
